### Root for main web page

#### Compile using:

```deno run -A build.ts```

#### Compile and serve:

```deno run -A build.ts --serve```

#### Run all tests:

```deno test .```

#### Build for production:

```deno run -A build.ts --prod```

##### Uses matrix math library called [gl-matrix](https://github.com/toji/gl-matrix)
